About

This site covers my life and work. It is designed to work by opening index.html directly from the filesystem - no web server required.

Everything uses relative paths and avoids JavaScript features that require a secure context.

Philosophy

The goal is to make authoring as simple as possible. Each page is just a script tag and body content. The script handles all the navigation, styling, and interactive features.

No build step is required. Just edit the HTML and refresh.

Structure

Each page lives in its own directory with an index.html file. Special directories prefixed with underscore hold scripts, assets, and derived files.

The sitemap is defined in the JavaScript file, making it easy to add new pages and have them appear in navigation automatically.

Features

The script injects a title from the first heading, adds navigation breadcrumbs for deeper pages, and creates a scrollspy in the right column for quick navigation within the page.

Top-level navigation links appear on every page, making it easy to jump between major sections.
